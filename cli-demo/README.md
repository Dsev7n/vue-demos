## vue-cli 脚手架
### 1.简介
+ 快速构建项目结构
+ 本身继承了多种项目模板

### 2.步骤
#### 2.1.安装：
+ npm install -g @vue/cli
+ vue -h

#### 2.2 初始化项目，生成项目模板
vue create +项目名称

#### 2.3 进入生成的项目目录，安装项目相关的依赖包
+ cd cli-demo
+ npm install  会自动安装

#### 2.4 运行
+ npm run serve   //启动测试服务
+ npm run build   //将项目进行打包输出dist目录，项目真正上线的话要将dist目录放到服务器上
